Presonus Faderport

Setup

Put the Faderport in Mackie Control mode:
- Switch on the Faderport
- Hold the stop button until the output button starts flashing
- Press the output button

Known issues

Unfortunately the faderport does fully support Mackie Control mode. Some functions and buttons behave slightly unexpected

Transport & window view

The transport buttons work as labeled.
Mix, Proj and Trns select the mixer, note editor and automation panels
Undo / Redo work as labeled
Loop works as labeled
Punch toggles overdub mode
User toggles the metronome
Mark toggles punch in

Fader & channel selection

The fader is connected to the currently selected channel in Bitwig. Change channels in the following ways:
- Click a channel in Bitwig
- Click 'bank' on the faderport and use the < and > buttons
- Output switches the fader to the master output
- Pan, mute, solo and rec work as labeled
Due to Faderport limitations the < and > buttons behave unexpected when the 'bank' button is not selected.

Fader mode

- Off switches the fader off
- Touch and write enable the arrange automation write and toggle either touch or write mode
- Read disables arrange automation writing
